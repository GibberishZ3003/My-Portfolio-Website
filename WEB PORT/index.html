About ME

ตั้งแต่วัยเด็กผมชื่อชอบในการสร้างแมพในเกม MINECRAFT ซึ่งมันไม่ใช่เพียงแค่การสร้างสรรค์ แต่ยังเป็นการใช้ความรู้ในด้านการใช้ Command Block เพื่อสร้างระบบที่ซับซ้อน จนถึงขั้นสามารถรับจ้างทำได้ และด้วยเหตุนี้ผมจึงได้ได้รับคำชมเชยและคำแนะนำให้มาพัฒนาทักษะด้านการเขียนโค้ดอย่างจริงจัง จึงทำให้ผมเริ่มศึกษาด้านนี้อย่างจริงจัง แล้วจึงพบว่ามีความชื่นชอบอย่างมากโดยเฉพาะ Data Science และ Web Development

ในด้าน Data Science ผมคิดว่าข้อมูลเป็นสิ่งสำคัญมากไม่ว่าจะทำงานไหนก็จำเป็นต้องการข้อมูลทั้งสิ้นเพื่อวิเคราะห์และประมวลผลข้อมูลเพื่อนำไปใช้ในการตัดสินใจอย่างมีประสิทธิภาพ ส่วนในด้าน Web Development ผมชอบในการคิดสร้างสรรค์และการออกแบบเว็บไซต์ สามารถสร้างสรรค์เว็ปไซต์ให้สวยงามและเป็นเอกลักษณ์ได้ตามความต้องการ ผมเชื่อมั่นว่าไม่ว่าจะเป็นใคร ทำงานอะไร อายุเท่าไหร่ แต่ในยุคดิจทัลนี้ทุกคนควรจะมีความรู้ในการเขียนโปรแกรมเป็นเบื้องต้นเพราะมันคือการช่วยเสริมทักษะและเพิ่มโอกาสในหลากหลายด้าน และมันจะกลายเป็นประโยชน์ในอนาคตอย่างแน่นอน
